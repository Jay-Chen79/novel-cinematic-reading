# 📖 Novel Cinematic Reading｜小说影视解说式沉浸阅读

> 像影视解说剪辑电影一样"剪辑"小说——保留原文最具冲击力的段落，用解说词串联，让读者获得远超自行阅读的沉浸体验。

## 这是什么？

一个 Claude Skill（提示词工程），用于将长篇小说转化为"影视解说式"的沉浸阅读体验。

你上传一段小说原文，它会像顶级影视解说 UP 主处理电影一样处理这段文字：挑出最炸的段落，用解说词串起来，控制节奏，埋伏笔，制造悬念——但绝不剧透，绝不替你思考。

**读者看到的 80% 是原文，解说词只占 20%。**

## 为什么做这个？

影视解说之所以比原片还爽，是因为它砍掉了铺垫，只留高潮，用解说词把名场面串成一条情绪过山车。

但书籍解读为什么总是干巴巴的？因为做书的人在"概括"，而不是在"剪辑"。他们把原文的味道抽干了，只剩下营养成分表。

**这个 Skill 的核心思路：不是总结书，而是"剪辑"书。** 原文负责所有的情绪和文字美感，解说者只负责调焦、对位、控制呼吸节奏。

## 核心特性

### 🎬 像剪辑电影一样剪辑小说
- 从原文中挑出"名场面"（情绪爆发、反转揭晓、文字高光、伏笔细节）
- 解说词只做穿针引线，原文完整保留不删不改

### 🔇 该闭嘴就闭嘴
- 当原文本身足够有力时，解说词完全退场
- 高潮段落不加任何评论，让读者直接面对原文冲击

### 🕳️ 挖坑必填
- 解说词埋下的每一个悬念，都会在后文回收
- 不留未兑现的伏笔

### 🎭 三种顶级解说风格融合

借鉴了三位华语顶级影视解说创作者的方法论：

| 风格 | 来源 | 用于 |
|------|------|------|
| **娓娓道来的深读者** | 木鱼水心（B站） | 背景铺垫、知识补充、结尾升华 |
| **情绪精准的节奏大师** | 毒舌电影（抖音） | 开头钩子、转折链条、悬念不断档 |
| **沉浸式场景还原者** | 心中之城（抖音） | 高潮退场、细节慢镜头、现实桥接 |

### 🎼 节奏控制
- 高潮-喘息-高潮的呼吸感
- 跨章节对照（把相隔很远的两段原文并排放置，不评论）
- 极简情绪标注（全篇最多 2-3 次，多了就变语文课）

## 使用方法

### 在 Claude.ai 中使用

1. 将 `SKILL.md` 的内容作为 System Prompt 或放入 Claude Project 的自定义指令中
2. 上传或粘贴小说原文
3. 对 Claude 说："按照 skill 生成沉浸式阅读体验"

### 输入

任意小说文本，支持：
- 单章 / 几个章节 / 整本书
- 纯文学 / 类型小说（悬疑、科幻、言情等）
- 中文 / 英文

### 输出

一份 Markdown 格式的沉浸式阅读文档：
- 原文段落以引用块（blockquote）呈现，标注 `📖 原文`
- 解说词以正常段落呈现
- 段落间用分割线分隔，制造场景切换感

## 效果示例

以鲁迅《狂人日记》为例——

**解说词（开头钩子）：**

> 1918年，一个叫鲁迅的人写了一篇小说。小说里有一个人发了疯，满嘴胡话，说所有人都想吃他。后来他"好了"，去做官了。
>
> 一百多年过去了，没有人能说清楚——他到底是疯了，还是醒了。

**解说词（伏笔埋设）：**

> "古久先生的陈年流水簿子"——什么东西又古又久，而且记了很多年的账？先记着这个问题。

**原文段落（高潮，解说完全退场）：**

> 我翻开历史一查，这历史没有年代，歪歪斜斜的每叶上都写着"仁义道德"几个字。我横竖睡不着，仔细看了半夜，才从字缝里看出字来，满本都写着两个字是"吃人"！

**解说词（伏笔回收）：**

> 那本"陈年流水簿子"，就是这部写满了"仁义道德"的历史。他踹了它一脚。所以所有人都容不下他。

**解说词（结尾，三种风格融合）：**

> 一个看穿了四千年吃人历史的人，最后"愈了"，加入了那个系统，成了那本陈年流水簿子上新添的一笔。
>
> 这是1918年的故事。你读完可以合上了。
>
> 也可以不合。

## 设计原则

1. **永远不要变成读书笔记或书评。** 解说词是为了增强阅读体验，不是为了分析文学技巧。
2. **永远不要替读者思考。** 点到为止，让读者自己发现、自己感受、自己顿悟。
3. **尊重原作。** 原文是主角，解说词是配角。
4. **绝不剧透。** 解说词制造的是"期待感"和"聚焦感"，不是"已知感"。
5. **口语化但不油腻。** 像一个真心喜欢这本书的朋友在给你安利。

## 适用场景

- 想快速体验一本长篇小说的精华
- 读过但想重新发现细节的经典作品
- 为读书会、播客、视频脚本准备素材
- 给朋友安利一本书时的"精华导读"

## 局限性

- 对短篇小说（<3000字）效果有限，原文本身已经足够紧凑
- 纯对话体小说需要更多背景补充，解说词占比可能略高
- 效果高度依赖原文质量——烂书剪不出好片
